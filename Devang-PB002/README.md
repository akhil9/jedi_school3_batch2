### Dependencies
Phone number validator
`npm`
`npm install commander`

Wikionterm python
`pip`
`pip install click`
`pip install urllib2`

Wikionterm ruby
`gem install wikipedia-client`

### Usage
Phone number validator
`npm link` in validate directory
`validate +919384747562`

Wikionterm python
`pip install --editable .` in wikionterm directory
`wikionterm --s Sachin --log log.txt`

Wikionterm ruby
`ruby wikionterm.rb`
